# Changelog

## v1.0.4

### Features

- **Task importance toggle**: Added a flag button next to each task to switch between normal and important status without re-adding the task.
- **Important task indicator**: Flag icon turns red and stays visible for important tasks, making priority at a glance.

### Changes

- **Removed glass density slider**: The Glass Density (Opacity) adjustment control has been removed from Settings. The glass theme still works with its default opacity.
- **Settings modal simplified**: Settings now only contain theme selection (Glass / Light / Dark).
