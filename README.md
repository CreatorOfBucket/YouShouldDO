## Downloads

Get the latest release from:

- GitHub Releases: https://github.com/CreatorOfBucket/YouShouldDO/releases

Latest Windows installer:

- **v1.0.3**: https://github.com/CreatorOfBucket/YouShouldDO/releases/tag/v1.0.3

### Windows (Recommended)

1. Open the Releases page and choose the latest version (e.g. v1.0.3).
2. Download and run: `YouShouldDO Setup 1.0.3.exe`.
3. Follow the installer wizard.

## Development

### 1. Requirements

Install [Node.js](https://nodejs.org/) (LTS recommended). `npm` is included.

### 2. Clone

```bash
git clone https://github.com/CreatorOfBucket/YouShouldDO.git
cd YouShouldDO
```

### 3. Install dependencies

```bash
npm install
```

### 4. Run

```bash
npm start
```

## Tech Stack

- **Framework**: Electron
- **Frontend**: HTML, CSS, Vanilla JavaScript (ES6+)

## Notes

- If `npm install` is slow in China, consider using a mirror (e.g. `npm config set registry https://registry.npmmirror.com`).
- Use a Node.js version compatible with the Electron version in this repo.
